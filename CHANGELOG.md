# Release Notes

v 0.0.1 (alpha) - 2026-01-xx

- Activitypub Core Functionality
- Allow for Multiple Actors: profile, followers and following
- Notes, Boosts, Replies & Quotes
- Inbox Stream
- Object Json Viewer to see activitypub json, for geeks like me
- Inbox Processing: configure batch size and processing interval, along with signature and block list verification
- Outpux Processing: configure batch size and processing interval
- Quote Permissions: setting to allow your items to be quoted
- Blocklist: domain based block list to tweak as you like
- Activity visibility - exposes all fringe activities that come in that can't be processes... yet.
- Clearing external activities and object - this isn't intended to store other peoples objects and activities forever. Set how long you want to keep external activities and objects.
